# Nostrlytics

## Visualize your Nostr data.

### Running Locally

```
yarn
yarn run dev
```

Then go to `http://localhost:5173`

### Completed so far

- [x] Bar chart of events by `kind`
- [x] Allow connect by hex pubkey, `npub`, or `nip-05`
- [x] Get raw event data from UI
